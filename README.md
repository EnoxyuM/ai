### Features
* **Easy GIF tutorial how to get API KEY:** You can add as many API as you want (Gemini only currently).
* **Local Storage:** All your chat history is saved locally in your browser.
* **Flexible Chat Management:** Includes features like web search, sending files, copying, branching conversations, regeneration, just like in AI Studio.
* **Model Selection:** 5 different models, including **Gemini 3.5 Flash**.
* **Mobile-Friendly:** Works on desktop and mobile devices.

### Возможности
* **Использование без ВПН:** Все ваши запросы отправляются не напрямую а в Cloudflare.
* **Изи GIF-инструкция по получению API ключа:** Можно добавлять столько ключей API, сколько захотите (пока только для Gemini).
* **Локальное хранилище:** Вся история ваших чатов сохраняется локально в браузере.
* **Функционал:** Веб-поиск, отправка файлов, копирование, создание веток диалогов и регенерация ответов, всё как в AI Studio.
* **Выбор моделей:** Доступно 5 разных моделей, включая **Gemini 3.5 Flash**.
* **Мобильная адаптация:** Отлично работает как на ПК, так и на мобилках.
